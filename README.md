## ✨ Technical Highlights

This portfolio was built entirely from scratch to demonstrate a strong foundation in core web technologies, performance optimization, and responsive design.

* **Zero-Dependency Architecture:** Built exclusively with pure HTML, CSS, and Vanilla JavaScript for maximum performance, yielding near-instant load times.
* **Modern CSS Design:** Heavily utilizes native CSS variables (`:root`), Flexbox, CSS Grid, and custom keyframe animations.
* **Interactive UI/UX:** * Custom typewriter logic built from scratch.
  * Highly performant scroll-reveal animations leveraging the native `IntersectionObserver` API.
  * Dynamic navigation highlighting based on active scroll sections.
* **Fully Responsive:** Implements fluid typography (using `clamp()`) and strict media queries to ensure a seamless layout from ultra-wide desktop monitors down to mobile devices.

## 🛠️ Tech Stack

* **Structure:** Semantic HTML5
* **Styling:** CSS3 (Variables, Grid, Animations, Responsive Design)
* **Interactivity:** Vanilla JavaScript (ES6+, DOM Manipulation, Web APIs)
* **Deployment & CI/CD:** Netlify
